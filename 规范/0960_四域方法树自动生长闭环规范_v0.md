# 0960 任务方法与任务树自动生长闭环规范 v0（保留原文件名兼容）

## 1. 核心闭环

当前自动生长闭环统一固定为：

```text
确定目标
-> 筹办完善任务树
-> 执行当前步骤
-> 结果写回结果节点
-> 比对目标
-> 未达成则回到筹办
-> 达成则完成
```

## 2. 三类生长对象

### 2.1 普通任务

- 生长目标：现实存在的目标特征值

### 2.2 任务管理任务

- 生长目标：宿主任务完成

### 2.3 学习任务

- 生长目标：目标方法可用


## 3. 筹办如何驱动自动生长

筹办自动生长只允许做：

1. 补当前步骤。
2. 补结果节点。
3. 补子任务。
4. 补方法实例条件节点与结果节点。
5. 必要时补新的虚拟存在特征类型，用来表达当前尚无字段可承载的结构缺口。


## 4. 执行如何驱动自动生长

执行自动生长只允许做：

1. 执行当前步骤。
2. 产出当前结果。
3. 触发下一轮筹办继续长树。

执行本身不直接发明新的主流程。
