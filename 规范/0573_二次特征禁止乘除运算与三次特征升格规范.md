# 0573 二次特征禁止乘除运算与三次特征升格规范

补充说明：

- 文件名沿用旧命名，便于兼容既有引用。
- 自 [0575_二次特征泛称与n次特征递推生成规范.md](D:\鱼巢\规范\0575_二次特征泛称与n次特征递推生成规范.md) 与 [0578_理解环与治理特征空间生长规范.md](D:\鱼巢\规范\0578_理解环与治理特征空间生长规范.md) 生效后，本文只约束 `比较型方向特征` 这条子链的数值边界。

## 1. 目标

本文用于固定以下边界：

- 哪些 `比较型方向特征` 必须保持 `显式比较 + 相减 + 区间`
- 何时不能把乘除结果继续伪装成“方向差值特征”
- 哪些更宽口径的高阶特征不受本文限制


## 2. 核心结论

必须固定：

1. 本文不再约束全部 `2次- n次特征族`。
2. 本文只约束 `比较型方向特征` 这一子族。
3. 对这条子族，正式数值母式仍应优先保持：

```text
左值
右值
差值 = 右值 - 左值
方向区间 = 正 / 零 / 负
```

4. 若某个结果需要通过 `乘法 / 除法` 才成立，它就不应继续冒充这条 `比较型方向特征`。
5. 但这不等于它不能成为别的合法高阶特征；聚合型、关系型、模式型、权重组合型特征允许采用各自母式。


## 3. 本文适用范围

本文只覆盖以下场景：

- 目标值与当前值比较
- 前后状态比较
- 左右对象同维度比较
- 明确以方向差值为核心语义的二次特征

本文不直接覆盖：

- 失败簇模式
- 复用价值判断
- 风险压力指数
- 权重组合后的治理解释
- 学习必要性评分
- 系统停滞指数

这些更宽口径结果，按 `0575 / 0578` 的高阶特征空间规则解释。


## 4. 对比较型方向特征的硬规则

必须固定：

1. 比较型方向特征必须显式可回指 `左值 / 右值 / 差值`。
2. 它的方向必须来自 `差值` 落入 `正区间 / 零区间 / 负区间`。
3. 不应把 `差值 ÷ 时间`、`差值 × 权重`、`差值 ÷ 量程` 再继续叫成同一条比较型方向特征。

一句话收口：

`比较型方向特征仍保持减法语义，但这不再构成全部高阶特征的总边界。`


## 5. 对旧口径的修订

以下旧口径自本文生效后废止：

1. `0573` 约束全部 `2次- n次特征族`
2. `变化率 / 加权值 / 归一化值` 一律不属于任何高阶特征
3. 一旦出现乘除就只能退到特征空间之外

修订后的统一口径是：

- 对 `比较型方向特征`：仍然优先使用减法与区间
- 对 `聚合型 / 关系型 / 模式型 / 权重组合型` 高阶特征：允许采用其合法母式，但必须显式说明来源与组合规则


## 6. 最终收口

`0573` 现在只收口比较型方向特征的数值边界；它不再否定更宽口径的高阶特征空间生长。凡需要通过权重组合、关系压缩、模式归纳或系统级聚合才能成立的结果，应直接作为合法高阶特征处理，而不是被旧口径误删。`
