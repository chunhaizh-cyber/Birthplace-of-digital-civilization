# 0565 判定依据分层与裁决顺序规范

## 1. 目标

本文用于统一回答以下问题：

- `一次特征 / 显式键 / 条件判定结果 / 二次特征 / 最终裁决` 五层各自是什么
- 哪一层能做什么，不能做什么
- 判定链路允许怎么流，不允许怎么跳
- 为什么最终裁决不能直接建立在一次特征或显式键之上

## 2. 五层总表

| 层级 | 名称 | 本质 | 允许职责 | 禁止职责 |
| --- | --- | --- | --- | --- |
| 第 1 层 | 一次特征层 | 原始真值输入面 | 承载值、状态、返回结果、原始事实 | 直接做最终裁决 |
| 第 2 层 | 显式键辅助层 | 显式字段、主键、条件主键、稳定ID、摘要字段 | 索引、召回、归桶、去重、摘要、桥接 | 直接做语义判定 |
| 第 3 层 | 条件判定结果层 | 比较后得到的条件类二次判定 | 表达可获取、可执行、缺口类型、条件满足与否 | 冒充原始真值或跳过比较过程 |
| 第 4 层 | 二次特征层 | 比较后得到的结构化关系结果 | 表达方向、对齐、偏离、稳定性、关系差值 | 被显式键或原始值越权替代 |
| 第 5 层 | 最终裁决层 | 外层综合结论 | 继续、停止、待机、收束、升格、回流、下一步去向 | 直接脱离第 3/4 层独立拍结论 |

更细的 `真值面 / 判定面 / 治理面 / 裁决面` 四类“面”的分工，按 [0572_真值面判定面治理面裁决面分工规范.md](D:\鱼巢\规范\0572_真值面判定面治理面裁决面分工规范.md) 执行。


## 3. 各层定义

### 3.1 第 1 层：一次特征层

一次特征层至少包括：

- `当前值`
- `目标值`
- `返回值`
- `原始状态`
- `原始动作结果`
- `原始布尔量`
- `原始数值`

它回答的是：

- `现在有什么`
- `现在是多少`
- `动作后读到了什么`

它不回答：

- `是否成立`
- `是否继续`
- `是否满足`
- `方向是否一致`

必须固定：

- 一次特征层是所有后续比较的真值输入面。
- 一次特征层本身不直接构成最终裁决。


### 3.2 第 2 层：显式键辅助层

显式键辅助层至少包括：

- `需求主键`
- `目标状态主键 / 目标特征类型`
- `方法主键 / 动作键`
- `输入条件主键`
- `结果节点主键 / 最近实际结果节点`
- `过程节点主键 / 动态稳定键`
- `路径主键`
- `锚点主键 / 状态面类型`

它回答的是：

- `哪些对象大致属于同一类`
- `哪些候选值得召回`
- `哪些证据适合先归到一起`

它不回答：

- `需求是否满足`
- `方法是否正确`
- `结果是否成立`
- `是否应该继续推进`

必须固定：

- 显式键只能帮助系统“更快找到候选”。
- 显式键不能越权直接替代比较和裁决。


### 3.3 第 3 层：条件判定结果层

条件判定结果层指的是：

- 在已有一次特征、状态模板、权限/可获取规则、条件展开规则基础上，
- 通过比较、展开、筛选后，
- 得到的条件类结构化结论。

典型例子包括：

- `可获取 / 不可获取`
- `有权限 / 无权限`
- `条件满足 / 条件不满足`
- `可直接执行 / 不可直接执行`
- `当前缺口类型`
- `当前下一步去向候选`
- `当前功能域候选`

必须固定：

- 条件判定结果不是原始布尔量，而是比较后的结果。
- 条件判定结果应视为 `面向条件语义的二次判定结果`。
- 因此它可以进入裁决层，但不能被误写成“绕开二次特征的另一套真值体系”。
- 在运行时实现上，条件判定结果必须通过 `原语输出` 形式进入后续选择链，不再允许直接裸读二次特征替代这一层。
- `可获取 / 有权限 / 条件满足 / 缺口类型 / 下一步去向` 这类结果的具体分类与生成顺序，按 [0566_条件判定结果层与条件裁决边界规范.md](D:\鱼巢\规范\0566_条件判定结果层与条件裁决边界规范.md) 执行。


### 3.4 第 4 层：二次特征层

二次特征层指的是：

- 由 `目标值 - 当前值`
- 或 `返回值 - 当前值`
- 或状态与状态、动态与动态之间的结构化比较
- 得到的关系结果。

典型例子包括：

- `需求方向`
- `结果方向`
- `方向一致 / 不一致`
- `绝对对齐 / 偏离`
- `关系差值`
- `稳定性方向`
- `根需求投影结果`

必须固定：

- 二次特征层是运行时最核心的正式证据层。
- 二次特征层回答“比较后形成了什么方向和关系”，不直接回答“下一步动作怎么选”。
- 任何“是否同向 / 是否成立 / 是否继续”的判断，都必须能回到二次特征层取证，但运行时不得裸判 `二次特征`，必须先经 [0577_原语层与选择函数规范.md](D:\鱼巢\规范\0577_原语层与选择函数规范.md) 定义的 `原语层` 转译。
- 当出现多个同向二次特征候选时，继续优选谁更接近需求，不再属于二次层本身，而属于 `3次 / n次特征` 的排序层。


### 3.5 第 5 层：最终裁决层

最终裁决层指的是外层真正要执行的结论，例如：

- `允许继续推进`
- `停止`
- `待机`
- `进入收束`
- `回到筹办`
- `转入学习`
- `允许升格`
- `允许回流`

必须固定：

- 最终裁决不得直接建立在第 1 层或第 2 层之上。
- 最终裁决必须建立在第 3 层和第 4 层的结果之上，但其运行时直接输入应统一视为 `原语输出`。
- 选择函数只能消费原语输出，不得绕回裸 `二次特征节点` 直接拍板。
- 若某场景没有形成第 3/4 层结果，则最多只能停留在“待进一步比较/待进一步观察/候选未定”。
- `继续 / 停止 / 待机 / 收束 / 允许转入学习 / 允许升格 / 允许回流` 的统一边界，按 [0567_最终裁决层与外层重判边界规范.md](D:\鱼巢\规范\0567_最终裁决层与外层重判边界规范.md) 执行。


## 4. 允许流向

当前规范允许的标准流向应固定为：

```text
一次特征
-> 比较/展开/筛选
-> 二次特征
-> 原语输出
-> 选择/最终裁决
```

这里必须补充固定：

- `0566` 中的条件判定结果，在运行时实现上统一视为 `原语输出` 的一种。
- `二次特征` 是证据层，不再允许直接从 `二次特征 -> 最终裁决` 裸跳。

显式键的正确位置应固定为：

```text
显式键
-> 候选召回/归桶/桥接
-> 辅助进入比较
-> 不直接下结论
```

必须固定：

1. 第 1 层可以输入到第 3 层和第 4 层。
2. 第 2 层只能辅助第 3/4 层形成，不能直通第 5 层。
3. 第 3 层和第 4 层进入第 5 层前，运行时必须先整理成 `原语输出`。
4. 第 5 层一旦形成后，若还会影响继续/停止/收束等根性行为，仍必须回外层重判。


## 5. 禁止流向

以下流向统一禁止：

1. `一次特征 -> 最终裁决`
2. `原始返回值 -> 最终裁决`
3. `显式键 -> 最终裁决`
4. `主键命中 -> 需求已满足`
5. `结果节点或结果摘要命中 -> 目标已完成`
6. `关系主键相同 -> 语义已经成立`
7. `状态面类型相同 -> 因果必然合法`
8. `方法自报成功 -> 任务推进已成立`
9. `裸二次特征节点 -> 直接继续/切换/终止`

一句话收口：

`原始输入不能跳过比较，辅助键不能跳过语义，二次特征不能跳过原语层。`


## 6. 三个典型例子

### 6.1 需求是否满足

正确顺序：

`需求值 / 当前值 / 返回值`
-> `需求方向 / 结果方向`
-> 比较两者是否一致
-> 得到 `推进成立 / 不成立`

错误顺序：

`结果节点或结果摘要命中`
-> `需求已满足`


### 6.2 方法是否可执行

正确顺序：

`当前状态 / 条件模板 / 权限状态 / 可获取状态`
-> `条件判定结果`
-> `可直接执行 / 需补条件 / 不可执行`

错误顺序：

`某个原始布尔量非零`
-> `可执行`


### 6.3 任务管理当前下一步去向

正确顺序：

`宿主状态 / 管理上下文 / 条件判定结果 / 二次特征`
-> `当前缺口类型`
-> `当前下一步去向`

错误顺序：

`当前功能域命中过去常用条件主键或摘要字段`
-> `直接决定下一步去向`


## 7. 与“只能使用二次特征”的关系

当前必须把这句话收严理解为：

`最终裁决不能建立在原始输入之上，必须建立在比较后形成的结果之上。`

因此：

- `一般方向判定` 应直接使用二次特征。
- `条件满足/缺口/可执行性` 这类条件类判定，应理解为 `条件语义上的二次判定结果`。
- 两者都属于“比较后产物”，可以进入最终裁决层。

但以下东西仍不算“比较后产物”：

- 原始主键
- 原始摘要字段
- 原始值
- 原始状态
- 原始返回值
- 方法自报成功


## 8. 当前阶段的统一要求

后续规范与实现都应统一满足：

1. 一次特征只作输入，不作终判。
2. 显式键只作辅助定位，不作终判。
3. 条件判定结果必须可追溯到比较与条件展开过程。
4. 二次特征必须可追溯到明确的比较对象。
5. 最终裁决必须能说明自己依赖了哪组条件判定结果和哪组二次特征。


## 9. 最终结论

可收敛为一句话：

`一次特征给真值，显式键给候选，条件判定给条件结论，二次特征给方向结论，最终裁决只消费比较后的结果。`
