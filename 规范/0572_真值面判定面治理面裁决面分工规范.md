# 0572 真值面判定面治理面裁决面分工规范

## 1. 目标

本文用于把 `真值面 / 判定面 / 治理面 / 裁决面` 四类“面”的分工单独固定下来。

本文要回答的是：

- 这四类“面”分别回答什么问题
- 它们和 `一次特征 / 条件判定结果 / 二次特征 / 最终裁决` 的关系是什么
- 哪些对象应该落在哪一面
- 哪些混用方式必须禁止

## 3. 核心结论

必须固定以下结论：

1. `真值面` 回答 `现在真实是什么`。
2. `判定面` 回答 `比较后得出了什么判断`。
3. `治理面` 回答 `下一轮治理姿态应如何调整`。
4. `裁决面` 回答 `这一轮最终准不准这样走`。
5. 四类“面”不得混层，不得互相越权替代。

一句话收口：

`真值面给真实输入，判定面给比较结论，治理面给后续调节入口，裁决面给本轮最终准入与去向。`


## 4. 四类“面”的定义

### 4.1 真值面

真值面承载原始真实输入。

它至少包括：

- `当前值`
- `目标值`
- `返回值`
- `原始状态`
- `原始结果节点`
- `原始指针/挂点`

它回答的是：

- `现在有什么`
- `现在是多少`
- `执行后实际回来了什么`

它不回答：

- `是否满足`
- `是否同向`
- `下一轮该怎么调`
- `最终准不准执行`


### 4.2 判定面

判定面承载比较、展开、筛选之后得到的结构化结论。

它至少包括：

- `需求方向`
- `预测结果方向`
- `实际结果方向`
- `对齐/反向/无推进`
- `可获取/不可获取`
- `有权限/无权限`
- `当前缺口类型`
- `当前下一步去向候选`

它回答的是：

- `比较后是什么`
- `当前条件是否成立`
- `当前更像往哪走`

它不回答：

- `值本身真实是什么`
- `下一轮要把哪些治理压力写回到哪里`
- `这一轮最终就此拍板`


### 4.3 治理面

治理面承载会影响后续治理姿态的正式作用位。

它至少包括：

- `学习压力`
- `重试压力`
- `风险记忆`
- `方法谨慎度候选`
- `方法可靠度候选证据`
- `更保守治理路径建议`

它回答的是：

- `下一轮更该学还是更该重试`
- `哪条路径更该谨慎`
- `哪些风险需要保留记忆`

它不回答：

- `当前真实值是什么`
- `当前需求是否已经满足`
- `这一轮最终是否允许继续`


### 4.4 裁决面

裁决面承载外层最终结论。

它至少包括：

- `是否允许继续推进`
- `是否允许当前执行`
- `是否待机`
- `是否停止`
- `是否收束`
- `是否允许回流`
- `是否允许升格`

它回答的是：

- `这一轮最终准不准这样走`
- `当前最终去向是什么`

它不回答：

- `真实输入值是多少`
- `条件为什么被判成某类`
- `经验项后果该写回哪个治理面`


## 5. 与五层判定链的关系

必须固定：

1. `真值面` 主要落在 [0565_判定依据分层与裁决顺序规范.md](D:\鱼巢\规范\0565_判定依据分层与裁决顺序规范.md) 的 `一次特征层`。
2. `判定面` 主要落在 `条件判定结果层` 与 `二次特征层`。
3. `治理面` 不等于五层中的某一层，它是后续治理写回与再调节的正式作用面。
4. `裁决面` 主要落在 `最终裁决层`。
5. `显式键辅助层` 只负责辅助定位与归桶，不单独构成这四类“面”的任何一面真值本体。


## 6. 当前阶段典型归类

| 对象 | 归属“面” | 说明 |
| --- | --- | --- |
| `安全值 / 服务值 / 当前值 / 返回值 / 结果节点指针` | 真值面 | 原始输入或原始回流结果 |
| `需求方向 / 实际结果方向 / 缺口类型 / 条件满足性 / 下一步去向候选` | 判定面 | 必须来自比较、展开或筛选 |
| `学习压力 / 重试压力 / 风险记忆 / 谨慎度候选` | 治理面 | 用来影响后续治理姿态 |
| `是否允许继续推进 / 是否允许执行 / 待机 / 停止 / 收束 / 允许回流 / 允许升格` | 裁决面 | 本轮最终结论 |


## 7. 必须禁止的混层

必须禁止以下写法：

1. 把 `安全值 / 服务值 / 返回值` 直接当成最终裁决。
2. 把 `当前下一步去向候选` 直接当成最终去向。
3. 把 `学习压力 / 风险记忆 / 谨慎度候选` 直接当成需求是否满足。
4. 把 `是否允许继续推进` 倒过来当成原始真值。
5. 把 `主键字段组合` 直接冒充真值面、判定面、治理面或裁决面。

一句话收口：

`真值面不能直接拍板，判定面不能越级终判，治理面不能冒充真值和裁决，裁决面不能倒灌成原始真值。`
