# 控制面板摘要线程二期完善计划 v0.1

## 1. 目标

控制面板摘要线程已经独立存在。本计划负责二期完善：

```text
接入真实世界观测版本。
接入运行事实摘要。
建立只读边界检查。
避免把占位文本伪装成正常摘要。
```

## 2. 子计划

### P1 世界版本来源

目标：

```text
摘要快照记录来源世界版本。
世界版本不能固定为 0。
```

验收：

```text
快照.世界版本 来自世界模型版本接口或只读观测版本。
快照.世界版本值 同步到摘要世界版本特征槽。
```

### P2 运行事实摘要

目标：

```text
摘要线程只读采集运行事实。
先接入方法树和任务树中的近期运行统计，再逐步接入运行虚拟存在索引。
```

摘要内容：

```text
有运行证据的方法数量
最近执行方法
最近执行时间
最近完成时间
失败任务数量
等待任务数量
运行虚拟存在索引是否接入
```

验收：

```text
运行事实摘要不再是占位文本。
未接入部分必须明确写为未接入。
```

### P3 只读边界检查

目标：

```text
摘要线程只读世界树、需求树、任务树、方法树和运行事实树。
```

禁止：

```text
取或创建
确保
注册
补齐
升格
修正
写世界树
写需求树
写任务树
写方法树
```

允许：

```text
写线程本地快照。
同步快照结构中的抽象特征指针和值槽。
```

验收：

```text
只读边界搜索无世界树写入命中。
若命中本地快照写入，应在验收记录中明确说明不是写主结构。
```

## 3. 完成口径

```text
摘要线程快照有非零世界版本。
运行事实摘要能反映方法和任务的近期运行情况。
摘要线程仍保持只读边界。
```
