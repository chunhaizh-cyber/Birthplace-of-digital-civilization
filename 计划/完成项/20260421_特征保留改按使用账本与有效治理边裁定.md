# 2026-04-21 特征保留改按使用账本与有效治理边裁定

## 本次完成

本次先完成规范修订，把“特征是否保留”的正式口径从“抽象价值评估优先”改成“使用事实优先”。

新的核心结论是：

- `特征保留不看抽象价值，只看有效参与治理闭环的使用事实`
- `被读取 / 被计算 / 被展示` 都不自动等于 `被使用`
- `有效使用` 的正式判断以 `有效治理边` 与 `使用事件/暴露事件` 为准
- `试用暴露期` 内的候选特征允许暂存
- `被使用但误导` 的特征不直接删除，而是转入修复或学习

## 这次固定下来的最小规则

1. 从未被使用，且试用期结束：归档。

4. 被使用但结果经常不好：不删除，转修复/学习。
