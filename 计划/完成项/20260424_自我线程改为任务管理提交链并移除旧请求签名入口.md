# 2026-04-24 自我线程改为任务管理提交链并移除旧请求签名入口

## 本次变更

- 自我线程当前主链不再继续使用旧的 `按需求创建任务头 / 刷新需求列表并确保需求相关任务 / 提交任务信息给任务管理任务` 函数名
- 新建并切换到一组明确的新入口：
  - `私有_按需求创建任务头_任务管理提交链(...)`
  - `私有_刷新需求列表并确保需求相关任务_任务管理提交链(...)`
  - `私有_提交任务信息给任务管理任务_任务管理提交链(...)`
- `私有_提交任务信息给任务管理任务_任务管理提交链(...)` 的入口面不再暴露旧 `结构_任务管理请求`、旧 `上下文` 作为函数签名
- 新入口面改为：
  - `结构_任务管理任务提交通道输入`
  - `结构_任务信息提交结果`
  - `结构_任务根节点 / 结构_任务控制态 / 结构_任务局部运行态`

## 语义收口

- 自我线程负责：
  - 按需求创建任务
  - 形成任务提交链输入
  - 将任务信息提交给任务管理任务
- 自我线程不再负责：
  - 在创建任务时同步目标方法
  - 通过旧 `请求/上下文` 入口暴露任务管理内部桥接结构

## 兼容说明

- 当前底层仍保留旧 `结构_任务管理请求` 作为内部兼容桥接层
- 但它已经退到新提交通道函数体内部，不再作为自我线程当前主链的入口签名
