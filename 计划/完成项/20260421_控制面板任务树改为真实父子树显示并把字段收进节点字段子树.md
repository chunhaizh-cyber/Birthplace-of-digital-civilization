# 控制面板任务树改为真实父子树显示并把字段收进节点字段子树

## 本次完成

控制面板之前虽然已经能显示任务树，但每个任务节点下面先挂的是一层：

`树下节点(n)`

真实的步骤节点、结果节点、子任务头结点都被包在这层壳里。

这样看起来更像：

`任务节点 -> 树下节点 -> 真实子节点`

而不是直接按真实任务树关系显示。

这次把这层中间壳去掉了，改成：

- 真实子节点直接挂在父任务节点下面
- 节点自己的详细字段收进单独的 `节点字段` 子树

也就是现在的结构是：

`任务节点 -> 真实子节点... -> 节点字段`

对应修改文件：

- `D:\鱼巢\控制面板类.cpp`

## 现在的显示语义

现在控制面板里的“任务树”页会先忠实显示真实任务结构：

- 头结点
- 步骤节点
- 结果节点
- 后续步骤/派生子任务

这些节点都直接按照父子关系展开。

每个节点自己的详细字段信息，则继续挂在该节点下的：

- `节点字段`

里面再细分为：

- `基本信息`
- `运行信息`
- `关系锚点`
- `头结点字段 / 步骤节点字段 / 结果节点字段`
