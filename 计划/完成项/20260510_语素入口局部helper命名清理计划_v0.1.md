# 语素入口局部 helper 命名清理计划 v0.1

日期：2026-05-10

## 1. 背景

前两轮已经完成：

```text
词性节点类 -> 语素入口节点类
词性主信息类 -> 语素入口主信息类
泛化基础信息绑定 API -> 语素入口命名
```

当前源码中仍有若干局部工具函数使用旧名：

```text
词性相同
词性同一
词性主键相等
词性主键
读取输出词性指针
读取包词性指针
```

这些函数实际比较或读取的是：

```text
语素入口节点指针 / 语素入口主键
```

不是人类词性枚举。

## 2. 命名边界

允许保留：

```text
枚举_词性
枚举_词性_工厂
人类词性
添加词性词
查找人类词性入口节点
绑定人类词性入口到场景基础信息
```

需要迁移：

```text
词性相同 -> 语素入口相同
词性同一 -> 语素入口同一
词性主键相等 -> 语素入口主键相等
词性主键 -> 语素入口主键
读取输出词性指针 -> 读取输出语素入口指针
读取执行输出词性指针 -> 读取执行输出语素入口指针
读取包词性指针 -> 读取包语素入口指针
输出词性 -> 输出语素入口
```

## 3. 实施步骤

```text
P1. 新增本计划。
P2. 迁移入口比较 helper 命名。
P3. 迁移入口指针读取 helper 命名。
P4. 保留人类词性入口 API，不误改。
P5. 检查旧 helper 名称残留。
P6. Debug x64 非增量构建验证。
P7. 更新计划索引。
```

## 4. 完成记录

```text
已完成。

完成内容：
```

## 5. 最终口径

```text
“词性”只用于人类语言标注；
语义节点比较和指针读取统一称为“语素入口”。
```
