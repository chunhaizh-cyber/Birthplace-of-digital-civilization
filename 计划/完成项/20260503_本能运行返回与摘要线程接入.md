# 20260503 本能运行返回与摘要线程接入

## 背景

继续执行 `20260503_自我线程瘦身与五线程分责迁移计划 v0.1`，同时处理本能函数 review findings。

本轮目标是：

```text
修正 改变量 可选与运行事实回写；
让 本能动作管理类 返回运行虚拟存在；
让 控制面板 消费 控制面板摘要线程 快照。
```

## 已完成

### 本能函数修正

```text
改变量 在旧兼容变化输入中改为可选，缺失时按 0 处理。
改变 I64 特征方法规格中 改变量 形参改为非必需。
旧兼容变化输出向方法虚拟存在补写 执行前值 / 执行后值。
差值驱动的 改变特征值 成功输出向方法虚拟存在同步完整执行摘要。
```

### 本能动作管理类修正

```text
本能动作函数 返回类型收敛为 存在节点类*。
检查是否可调用 与 执行功能函数 返回 存在节点类*。
运行结果不再是裸特征节点，而是按运行开始时间唯一键创建的运行虚拟存在式结果。
运行结果写入 本能动作运行结果码 与 方法执行情况。
执行真实本能函数后，会给返回的运行存在补写 本能动作运行结果码。
```

### 控制面板摘要线程接入

```text
控制面板快照新增 控制面板摘要线程 状态与摘要字段。
控制面板摘要文本新增 摘要线程 行。
线程状态树新增 控制面板摘要线程 节点。
节点详情 JSON 支持 thread-summary。
读取控制面板快照时启动摘要线程、请求刷新，并最多等待 100ms 首帧。
```
