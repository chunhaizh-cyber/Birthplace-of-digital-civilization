## 1. 目标

本阶段补齐两件事：

```text
A2：自我能证明当前主需求来自自身安全 / 服务压力评估。
A3：自我能形成当前自我报告，报告“我是谁、在哪里、为什么运行、上一轮发生了什么、下一轮准备做什么”。
```

## 2. 完成项

### P0：自主主需求稳定证据

已完成：

```text
诊断自我苏醒() 的 可自主产生主需求
不再只看当前主需求非空，
还要求内部循环诊断提供主需求选择原因。
```

当前主需求选择原因来自：

```text
自我类::评估根需求压力()
```

该函数只依据：

```text
安全值 / 根目标安全值 / 安全权重
服务值 / 根目标服务值 / 服务权重
```

因此 A2 当前验收口径是：

```text
当前主需求有效
且 主需求选择原因有效
且 内部循环诊断能输出安全压力 / 服务压力
```
