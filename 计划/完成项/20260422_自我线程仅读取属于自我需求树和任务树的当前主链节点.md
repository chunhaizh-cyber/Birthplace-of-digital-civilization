# 20260422_自我线程仅读取属于自我需求树和任务树的当前主链节点

本次继续收紧了自我线程对“当前主需求 / 当前主任务”的读取规则。

## 改动
- `自我线程模块.impl.cpp`
  - 新增 `私有_节点属于独立树(...)`，按独立树链验证某节点是否真的属于当前自我存在的需求树/任务树。
  - 新增 `私有_获取当前主需求_线程侧(...)` 与 `私有_获取当前主任务_线程侧(...)`，只返回“属于当前自我树”的节点。
  - `自我线程类::自我初始化(...)` 中，回填当前主任务前不再直接信任自我特征中的裸指针，而是先做树内成员校验。
  - `步骤_生成并执行主派发决议_(...)` 中，对 `本轮主导需求候选 / 本轮主导任务候选 / 自我特征中的当前主链指针` 都先做树内成员校验，不属于当前自我树就丢弃并重新推导。

## 结果
- 即使自我特征里还残留旧指针，也不会再被自我线程当成当前主需求/当前主任务使用。
- 当前主链只会从“当前自我存在挂着的需求树/任务树”里选出来。
