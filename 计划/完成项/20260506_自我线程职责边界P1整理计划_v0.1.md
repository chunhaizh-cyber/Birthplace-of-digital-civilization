# 20260506_自我线程职责边界P1整理计划_v0.1

## 目标

将 `自我线程模块.impl.cpp` 从“所有自我线程实现聚合文件”逐步整理为只承载生命周期、主循环、阶段推进的主干文件。

## 目标模块边界

| 模块 | 职责 |
| --- | --- |
| `自我线程模块.impl.cpp` | 生命周期、主循环、阶段推进、跨模块调度胶水 |
| `自我线程_消息邮箱模块.ixx` | 消息入队前排序规则、冻结/待消费队列摘要、邮箱优先级规则 |
| `自我线程_需求治理模块.ixx` | 需求树变更、需求队列刷新、主需求选择 |
| `自我线程_价值结算模块.ixx` | 安全值 / 服务值结算、任务反馈账本、去重与幂等 |
| `自我线程_任务派发模块.ixx` | 任务队列、任务包、工作线程桥接 |
| `自我线程_回流治理模块.ixx` | 结果回流、派生需求、父任务回接 |
| `自我线程_运行审计模块.ixx` | 阶段事件、摘要、因果/调试文本、通用审计工具 |

## 本轮完成

1. 已存在并接入 `自我线程_价值结算模块.ixx`，作为价值账本入口。
2. 新增 `自我线程_消息邮箱模块.ixx`：
   - `治理消息优先级值`
   - `治理消息处理类型文本`
   - `消息排在前面`
   - `双队列摘要`
3. 新增 `自我线程_运行审计模块.ixx`：
   - `布尔文本`
   - `文本或默认`
   - `特征值摘要`
   - `有界追加`
4. `自我线程模块.impl.cpp` 已改为通过上述模块转发邮箱排序和运行审计通用工具。
5. 保留兼容薄转发函数，避免一次性大范围改动主循环和回流路径。
