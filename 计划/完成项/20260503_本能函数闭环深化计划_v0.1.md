# 本能函数闭环深化计划 v0.1

## 1. 目标

本计划负责把线程分责成果接入真正的执行闭环。

主链定为：

```text
自我线程派发
  -> 工作线程筹办
  -> 查找方法返回选中自我能力值
  -> 配对参数
  -> 可执行输入参数场景
  -> 本能函数执行
  -> 运行事实回写
  -> 缺口继续生长需求树
```

## 2. 优先顺序

```text
查找方法
配对参数
改变特征值
尝试学习
查找特征类型
检查需求是否满足
```

## 3. 总规则

```text
本能函数不按名称做语义判断。
查找方法只判断 需求条件包 是否被 方法能力包 包含。
查找方法输出选中自我能力值。
任务长期保存自我能力枚举值，不保存方法首节点作为权威字段。
配对参数负责生成可执行输入参数场景。
执行本能函数只接受闭合输入参数场景。
失败原因尽量生成缺口需求，而不是静默失败。
```

新增语义字段必须同步设计抽象特征和实例特征槽。

## 4. 子计划

### P1 查找方法

目标：

```text
在限制包约束下判断 需求条件包 是否被 方法能力包 包含。
不使用方法名称、动作名称、显示文本、函数名参与匹配。
返回选中自我能力值。
```

输出：

```text
方法查找情况
选中自我能力值
能力配对结果存在
原始需求绑定包
匹配路径记录
类型抽象距离
值域抽象距离
总匹配距离
待配对形参表
```

验收：

```text
查找成功时输出自我能力枚举值。
方法首节点只可作为缓存输出。
原始需求口径必须保留，不能被父级能力口径替换。
```

### P2 配对参数

目标：

```text
读取方法配对结果、需求条件包、形参表和限制包。
把原始需求绑定到方法形参。
生成可执行输入参数场景。
```

失败输出：

```text
参数条件缺口
缺少执行差值
缺少目标执行后值
缺少目标实例特征
缺少当前值观测
限制包不允许
```

验收：

```text
参数配对成功后输出闭合的可执行输入参数场景。
参数配对失败后生成条件缺口需求。
不默认补执行差值、目标执行后值或当前值。
```

### P3 改变特征值

目标：

```text
从旧的 当前值 + 改变量 / 方向 模型切换为差值约束执行模型。
```

闭合输入：

```text
目标宿主
目标抽象特征
目标实例特征
目标值类型
差值约束包
执行差值 或 目标执行后值
实例缺失策略
溢出策略
限制包
```

执行规则：

```text
执行前值只能从目标宿主下的目标实例特征读取。
执行后值 = 执行前值 + 执行差值。
实际执行差值 = 执行后值 - 执行前值。
实际执行差值必须命中差值约束区间。
抽象特征不得被直接写当前值。
写入后触发抽象区间命中解析。
```

验收：

```text
不再从输入参数场景读取当前值作为计算依据。
执行差值和目标执行后值同时存在时必须一致。
I64 加法和差值计算有溢出保护。
成功和失败都写运行虚拟存在、输出结果场景和方法统计。
```

### P4 尝试学习

目标：

```text
让只有方法头节点或结构不完整的方法逐步长出可验证的方法结构。
```

两种模式：

```text
自检补全模式：
  补方法虚拟存在、候选能力包、候选形参包、候选结果包、候选条件结果对。

试验学习模式：
  根据形参包生成实参候选，试运行目标方法，收集运行事实，更新条件结果对和值域区间。
```

验收：

```text
方法只有头节点时不伪造运行结果。
没有执行入口时只生成结构缺口或执行体缺口。
有执行入口时可以根据形参包生成候选实参并试运行。
条件结果对可从候选、试用逐步升格为稳定。
```

### P5 查找特征类型

目标：

```text
为查找方法、配对参数、改变特征值提供抽象特征父链和限制包判断能力。
```

验收：

```text
能判断需求特征是否可上升到能力范围特征。
能记录抽象路径和抽象距离。
能检测超过最高允许特征类型或最大抽象层数。
```

### P6 检查需求是否满足

目标：

```text
执行后根据需求条件包和输出结果场景判断当前需求是否满足。
```

验收：

```text
满足时写需求满足证据。
不满足时生成结果缺口需求。
父需求可在子需求满足后重新筹办或执行。
```

## 5. 完成口径

```text
查找方法返回自我能力值。
配对参数生成可执行输入参数场景。
改变特征值只执行闭合输入场景。
当前值只能从目标实例特征读取。
尝试学习能根据方法参数包生成实参并更新条件结果对。
运行事实能反向驱动需求树缺口继续生长。
```
