# 本能函数执行闭环深化计划 v0.1

## 1. 目标

```text
让查找方法、配对参数、改变特征值、尝试学习进入运行时本能动作集。
让任务筹办和执行阶段能够通过自我能力值命中这些本能函数。
避免把自我动作实现模块反向接入自我初始化模块造成模块循环。
```

## 2. 范围

```text
查找方法
配对参数
改变特征值
尝试学习
```

本轮不扩展新的长期语义结构体或枚举。已有语义继续通过抽象特征与实例特征槽表达。

## 3. 执行内容

```text
1. 在自我动作实现模块中新增本能函数执行闭环注册入口。
2. 注册入口把四个本能函数写入本能动作运行时动作集。
3. 鱼巢主入口导入自我动作实现模块。
4. 自我环境初始化完成后刷新本能函数执行闭环注册。
5. 自我线程启动完成后刷新本能函数执行闭环注册。
6. 新注册入口不向自我初始化模块添加反向 import，避免模块依赖环。
7. 新增代码不引入调用点说明字符串。
```

## 4. 验收口径

```text
查找方法进入运行时动作集。
配对参数进入运行时动作集。
改变特征值进入运行时动作集。
尝试学习进入运行时动作集。
自我动作实现模块和自我初始化模块之间无新增模块循环。
Debug x64 构建通过。
```
