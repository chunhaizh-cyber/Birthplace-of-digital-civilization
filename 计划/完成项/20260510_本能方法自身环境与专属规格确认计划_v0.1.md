## 1. 定位

本计划用于把“每个本能方法执行前先确保自身方法环境”的规范落到代码。

当前规范已经明确：

```text
每个方法先把自己建完整，再处理目标。
```

其中：

```text
自身方法环境 =
    方法首节点
  + 方法虚拟存在
  + 本轮运行虚拟存在
  + 条件场景
  + 结果场景
  + 输入槽
  + 输出槽
  + 能力签名
  + 专属规格特征
```

## 2. 先后顺序

本计划排在以下工作之后：

```text
1. 方法主信息 variant 和方法虚拟存在服务已完成。
2. 任务筹办 / 任务执行已从本能方法迁出。
3. 信息入口类型与语素入口命名收口已完成。
4. 需求树生长本能转换规范已制定。
```

本计划排在以下工作之前：

```text
1. 学习需求正式入树。
2. 外界探索需求自动生成。
3. 学习 / 练习方法经验沉淀增强。
4. 本能转换方法注册扩大化。
```

原因：

```text
学习需求入树和方法经验沉淀都依赖学习类本能方法稳定暴露自身规格。
若本能方法自身规格不稳定，学习需求会不知道要补哪个方法、哪个槽、哪个结果。
```

## 3. 非目标

本计划不做：

```text
1. 不恢复任务筹办本能方法。
2. 不恢复任务执行一步本能方法。
3. 不让方法虚拟存在替代方法首节点。
4. 不新增任务 / 方法虚拟存在类。
5. 不自动生成学习需求。
6. 不生成外界探索需求。
7. 不用名称、动作名或显示文本做方法语义判断。
```

## 4. 分阶段计划

### M0：代码现状复核

目标：

```text
确认当前 `本能方法类`、`方法类`、`方法虚拟存在服务类` 和 `自我动作实现模块` 中已有的确保方法头、方法虚拟存在、规格确认函数位置。
```

动作：

```text
1. 搜索 `确保本能方法`、`本轮运行虚拟存在`、`方法虚拟存在`、`确认尝试学习规格`、`确认定向学习规格`、`确认练习规格`。
2. 标出通用逻辑和专属规格逻辑的重复段。
3. 记录无法统一的特殊方法。
```

验收：

```text
形成代码差距表。
```

### M1：定义通用环境结构

目标：

```text
新增或整理 `结构_本能方法自身环境`。
```

要求字段：

```text
方法首节点
方法虚拟存在
本轮运行虚拟存在
方法头已就绪
方法虚拟存在已就绪
方法规格已确认
失败原因
```

验收：

```text
结构可被学习 / 练习类本能方法复用。
```

### M2：实现统一确保入口

目标：

```text
实现 `确保本能方法自身环境`。
```

入口职责：

```text
1. 确保本能方法本体。
2. 确保本轮运行虚拟存在。
3. 确保方法虚拟存在。
4. 调用专属规格确认函数。
5. 失败时记录日志并报错，不静默兜底。
```

验收：

```text
任一环节失败时有错误日志和明确失败原因。
```

### M3：拆出专属规格确认入口

目标：

```text
新增 `确认本能方法专属规格` 分发入口。
```

分发：

```text
自我_尝试学习 -> 确认尝试学习规格
自我_定向学习 -> 确认定向学习规格
自我_练习 -> 确认练习规格
其他本能方法 -> 确认通用本能方法规格
```

验收：

```text
三类学习本能方法规格确认不再散落在函数体开头。
```

### M4：迁移学习类本能方法开头

目标：

```text
`尝试学习`、`定向学习`、`练习` 开头统一使用自身环境入口。
```

要求：

```text
1. 先确保自身方法环境。
2. 再读取输入条件包。
3. 再处理目标方法环境。
4. 目标方法缺口和自身环境失败分开记录。
```

验收：

```text
学习类方法能在控制面板或日志中区分：
    自身环境失败
    输入绑定失败
    目标方法缺口
    核心逻辑失败
```

### M5：补充规格槽清单

目标：

```text
按规范补齐尝试学习、定向学习、练习的输入槽、输出槽、条件槽和结果槽。
```

验收：

```text
三类学习方法的能力签名和规格槽能被方法查找 / 诊断读取。
```

### M6：验证

命令：

```text
msbuild .\鱼巢.vcxproj /p:Configuration=Debug /p:Platform=x64 /p:LinkIncremental=false /m
.\x64\Debug\鱼巢.exe --self-loop-summary
.\x64\Debug\鱼巢.exe --causal-learning-target-summary
.\x64\Debug\鱼巢.exe --causal-learning-demand-gate-summary
```

验收：

```text
构建通过；
现有摘要命令正常返回；
学习候选诊断不因自身规格缺失而静默降级；
规格缺失时有日志和报错。
```
