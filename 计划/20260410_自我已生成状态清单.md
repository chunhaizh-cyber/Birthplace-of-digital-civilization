## 2. 当前已生成的自我状态

当前已经真实生成的自我状态共 `8` 项，且都来自 `任务管理任务模块` 的治理证据链。

1. `任务管理_当前宿主任务指针`
   - 含义：当前治理轮选中的宿主任务。
   - 触发：宿主任务被选中或切换时生成。
   - 当前来源：`任务管理_宿主任务切换`

2. `任务管理_当前方法指针`
   - 含义：当前宿主任务绑定的方法挂点。
   - 触发：当前方法被绑定、补齐或切换时生成。
   - 当前来源：`任务管理_当前方法挂点`

3. `任务管理_宿主任务状态`
   - 含义：当前宿主任务的运行状态镜像。
   - 触发：宿主任务在 `未启动 / 挂起 / 运行中 / 完成 / 失败 / 取消` 等状态间迁移时生成。
   - 当前来源：`任务管理_宿主任务状态迁移`

4. `任务管理_当前功能域`
   - 含义：当前治理轮所处的功能域。
   - 触发：本轮治理确定自己处于 `筹办 / 维护 / 纠偏 / 收束` 等域时生成。
   - 当前来源：`任务管理_当前功能域`

5. `任务管理_当前缺口类型`
   - 含义：当前治理轮显性化出来的主缺口类型。
   - 触发：治理结果识别出当前主要缺口时生成。
   - 当前来源：`任务管理_当前缺口类型`

6. `任务管理_当前下一步去向`
   - 含义：当前治理轮建议的下一步工程去向。
   - 触发：治理结果给出 `进入运行 / 保持运行 / 绑定方法 / 回到重筹办 / 进入收束` 等去向时生成。
   - 当前来源：`任务管理_当前下一步去向`

7. `任务管理_当前步骤指针`
   - 含义：当前治理步骤节点的镜像指针。
   - 触发：步骤节点创建、复用或推进时生成。
   - 当前来源：`任务管理_当前步骤推进`

8. `任务管理_最近结果指针`
   - 含义：当前治理链上最近结果节点的镜像指针。
   - 触发：结果节点创建、复用或推进时生成。
   - 当前来源：`任务管理_最近结果推进`
